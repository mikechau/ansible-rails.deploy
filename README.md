# Rails Deploy Ansible Role

This is a role for deploying `Ruby on Rails` applications, for rolling upgrades via `Haproxy` and `Phusion Passenger`.

Currently it assumes you are using:

- CentOS
- Haproxy
- Phusion Passenger Standalone
- RVM
